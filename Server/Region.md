# 选择地区 {#Region}

## 香港(HK) {#ZH-TW}

香港目前提供了2个版本,分别是常规版本(带有`Normal`)与旧版支持(带有`Legacy`标记)。

### 如何选择? {#TW-Choose}

Normal版本支持最新版本以及大版本更新后的版本。\
而Legacy使用的是老版本服务,他支持更早的版本,但同时也就不支持新版本。
